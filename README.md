# 🎓 Student Information System (Java OOP Mini Project)

## 📌 Project Overview

This project is a **Student Information System** developed using **Java Object-Oriented Programming (OOP)** concepts.

The program allows users to:

* Enter multiple student details
* Store them using objects
* Display all student information

This project demonstrates the use of **classes, constructors, arrays of objects, static variables, loops, and user input using Scanner**.

---

## 🚀 Features

* Add multiple student records
* Store student data using objects
* Display all student details
* Uses static variable for common college name
* User input using Scanner
* Simple console-based application

---

## 🛠️ Technologies Used

* **Java** – Core programming language
* **OOP Concepts** – Classes, objects, constructors, static variables
* **Scanner Class** – User input handling

## ▶️ How to Run the Program

1. Clone the repository.
2. Open the project in any Java IDE or terminal.
3. Compile the program:

```
javac Main.java
```

4. Run the program:

```
java Main
```

---

## 📋 Program Workflow

1. The user enters the number of students.
2. The program collects student details:

   * Name
   * ID
   * Department
   * Age
3. Student objects are created and stored in an array.
4. The program displays all student information.
